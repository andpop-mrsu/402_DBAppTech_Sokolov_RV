# Игра "Холодно-Горячо"

Программа для консольной игры **"Холодно-Горячо"**.  
Игрок пытается угадать число, загаданное компьютером, компьютер загадывает **трёхзначное число без повторяющихся цифр**, а игрок пытается его угадать.

## [Играть онлайн](https://nevall-von-goodem.github.io/Cold-hot-web/)

## Как играть
1. Открой `index.html` локально в браузере **или** сайт GitHub Pages.
2. Введите имя игрока и нажмите «Начать».
3. Вводите трёхзначные числа без повторяющихся цифр (например, `123`).  
   Некорректный ввод не засчитывается.
4. Команда `exit` — досрочный выход (покажем секрет).
5. После окончания партия сохраняется в **IndexedDB**. Раздел «История» позволяет:
   - просмотреть список партий,
   - повторить любую (пошаговые ходы и подсказки),
   - очистить локальную БД.

### Работа с базой (IndexedDB)
– Список партий: раздел «История».  
– Повтор партии: нажмите «Повтор» в таблице или введите ID и нажмите «Повтор по ID».  
– Прямая ссылка: `#replay-<ID>` (например, `#replay-12`).  
– Очистить локальную БД: кнопка «Очистить БД» в разделе «История».
